# Biohazard Archive: Raccoon City Incident

An immersive, interactive 3D map documenting the history of biohazards in the *Resident Evil* universe. Explore character paths, analyze viral strains, and uncover the downfall of Umbrella Corporation.

## Features

*   **3D Interactive Globe**: Explore locations of major incidents across the globe using `react-globe.gl`.
*   **Virus Database**: Detailed information on viral strains (T-Virus, G-Virus, Las Plagas, etc.).
*   **Character Paths**: Track the movements and survival stories of iconic characters like Chris Redfield, Jill Valentine, and Leon S. Kennedy.
*   **Multi-language Support**: Fully localized interface in English, Italian, German, and French.
*   **Nostalgia Mode**: Toggle CRT/pixel-art effects for a retro-horror aesthetic.

## Tech Stack

*   **Framework**: [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
*   **3D Visualization**: [react-globe.gl](https://github.com/vasturiano/react-globe.gl) (powered by Three.js)
*   **Styling**: [Tailwind CSS](https://tailwindcss.com/)
*   **Animations**: [Framer Motion](https://www.framer.com/motion/)
*   **Internationalization**: [i18next](https://www.i18next.com/)
*   **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run development server**:
    ```bash
    npm run dev
    ```

3.  **Build for production**:
    ```bash
    npm run build
    ```
